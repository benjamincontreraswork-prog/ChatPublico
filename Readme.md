ChatPublico

Aplicación de comunicación publica en tiempo real basada en arquitectura cliente-servidor.

Proyecto para unificar conocimientos sobre arquitectura y diseño de sofrware

Objetivo (motivo de creación)
Crear una plataforma web de comunicación donde múltiples usuarios puedan interactuar mediante mensajes en un canal público, administrando identidad, permisos y persistencia de información.

Objetivos funcionales

permitir conexión desde navegador
crear identidad de usuario
enviar mensajes
modificar mensajes bajo reglas
eliminar mensajes según permisos
cerrar sesión

Tecnologías previstas:

Backend: C
Frontend: HTML, CSS, JS
Base de datos: SQLite
Comunicación: HTTP
Servidor: Linux
